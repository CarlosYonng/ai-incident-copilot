# Changelog

## 0.1.0 - 2026-06-16

Initial MVP release candidate.

- Added Spring Boot backend with Incident, Workflow, Action, Metrics, MCP audit, Runbook, and Postmortem modules.
- Added React + TypeScript frontend console for the complete incident workflow.
- Added Docker Compose, Flyway schema migration, demo scripts, smoke test, and unit tests.
- Added Diagnosis MCP JSON-RPC integration with configurable fallback mode.
- Added release-readiness documentation and security notes.
